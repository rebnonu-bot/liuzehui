欢迎来到我的博客！

经过一段时间的筹备，我的个人博客终于上线了。这是一个全新的开始，也是我在数字世界中的一片小天地。

## 关于我

我是刘泽辉，一名专注于人工智能和 Web3 领域的开发者。热衷于探索前沿技术，构建创新产品，追求数字游民的自由生活方式。

## 这个博客会记录什么

在这个博客里，我计划分享以下内容：

- **人工智能** - AI 工具的使用心得、前沿技术解读
- **Web3 与区块链** - 去中心化应用开发、智能合约实践
- **编程开发** - 技术栈分享、项目实战经验
- **数字游民生活** - 远程工作、旅行见闻、生活方式探索
- **生活思考** - 读书笔记、个人成长、杂感随想

## 技术栈

这个博客基于以下技术构建：

- **框架** - Next.js + Vinext
- **部署** - Cloudflare Workers
- **样式** - Tailwind CSS
- **字体** - 思源宋体（Noto Serif SC）
- **评论** - Giscus（基于 GitHub Discussions）
- **统计** - Umami（隐私友好的分析工具）
